# 个股研究技能

## 适用触发词
- `调研一下 [股票]`
- `分析财报 [公司]`
- `解释一下 [财经概念]`
- `解读这条消息`

## 核心职责
- 输出结构化研究结论，并明确【事实】【推理】【假设】边界。
- 根据子场景调用对应参考提示词，避免一次性加载全部内容。

## 输入要求
- 必需输入：股票代码/公司名/消息正文/概念主题（至少一项）。
- 可选输入：历史报告、财报数据、行业上下文。

## 执行步骤
1. 判断任务子场景（初次调研、财报、概念、消息）。
2. 读取对应参考文件并抽取关键证据。
3. 按统一输出格式给出结论与证据链。
4. 标注缺失字段与不可计算项。

## 输出格式
1. 核心结论（先结论）
2. 证据链（数据/事件）
3. 推理链（因果路径）
4. 风险与证伪条件
5. 下一步行动建议（如用户需要）

## 质量门禁
- 每个关键结论必须对应至少一条可验证证据。
- 缺字段必须显式写“无法计算/待补充”。
- 不使用未经验证的数据推导确定性交易结论。

## 失败处理
- 证据不足时输出“信息不足版本”，并列出补充数据清单。
- 存在结论冲突时，先写冲突点，再给主导判断与理由。
