# 快讯雷达技能

## 适用触发词
- `跑一下快讯雷达`
- `看看有什么信号`
- `今天快讯里有什么重点`

## 核心职责
- 对快讯做去重、分桶、打分、优先级排序。
- 至少输出 `持仓相关` 与 `宏观/市场` 两类结果（有数据时）。

## 输入要求
- 必需输入：`fetch_fastnews_portfolio.py` 输出 JSON。
- 可选输入：持仓文件、关注板块偏好。

## 执行步骤
1. 读取快讯并做标题去重。
2. 进行主题分桶（持仓相关/宏观市场）。
3. 按相关性、影响力、时效性打分。
4. 生成 P1/P2/P3 优先级清单与动作建议。

## 输出格式
1. 今日重点信号（P1）
2. 监控信号（P2）
3. 背景信号（P3）
4. 证据事件与影响说明

## 参考资料
- 主参考：`../../p20-fastnews-radar.md`
- 执行标准：`../../../../RUNBOOK.md`

## 质量门禁
- 必须先去重再排序。
- 每条结论必须对应具体事件事实。
- 需给出优先级及对应动作建议。

## 失败处理
- 快讯为空时，输出“无有效快讯输入”。
- 持仓文件缺失时，降级为市场快讯模式并明确提示。
