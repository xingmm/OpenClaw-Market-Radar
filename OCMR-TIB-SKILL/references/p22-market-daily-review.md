# P22 盘面每日复盘（趋势为王，结构修边）

## 一、输入与范围
- 主判据指数：上证指数（日线为主）
- 同步校验指数：深证成指、创业板指、上证50、沪深300、中证500
- 本模板用于每日复盘，不替代盘中交易系统。

## 二、趋势判读（日线）
- 短期趋势线：EXPMA(30)
- 长期趋势线：EXPMA(144)
- 趋势近距离阈值：
  - `near = |EXPMA30-EXPMA144| / EXPMA144 <= 2%`

输出要求：
- 当前价格相对EXPMA30/EXPMA144位置
- 是否破短期趋势、是否破长期趋势
- 短长趋势是否“很近”（<=2%）

## 三、结构判读（MACD）
周期：60m、90m、120m、日线

提示项（都要给）：
- 钝化（邻钝/隔钝）
- 结构形成（邻构/隔构）
- 钝化消失（邻消/隔消）
- 结构消失（邻消/隔消）

动作规则：
- 仅“结构形成”参与仓位动作。
- 不要求四周期联判；若多周期共振，额外标注“共振增强”。
- 结构有效窗口：对应周期后续24根K线。

## 四、TD9辅助（仅提示）
周期：月、周、日、120m、90m、60m

规则：
- TD9只做高低点提醒，不触发自动仓位动作。
- 低9在“无结构风险”时可作为低点参考提示。

## 五、仓位动作（总仓位口径）
按总仓位100%给出减仓层数。

1) 无结构：
- 破短期趋势 -> 减4层
- 再破长期趋势 -> 再减2层（累计6层）

2) 有结构：
- 若短长趋势很近（<=2%）且结构形成 -> 减6层
- 若价格近趋势但不满足高威胁 -> 减4层
- 若价格远趋势 -> 可不减，仅提示风险

## 六、固定输出模板
1. 趋势状态（日线）
- 数据：收盘、EXPMA30、EXPMA144、趋势距离
- 推导：是否破位，是否近趋势
- 结论：趋势风险等级

2. 结构状态（60m/90m/120m/日）
- 各周期：钝化/结构/消失状态 + 是否共振 + 剩余影响周期

3. 仓位动作建议（总仓位）
- 明确“减X层”与触发条件
- 如无动作，明确“维持仓位”的依据

4. TD9提示（仅提醒）
- 月/周/日/120/90/60 的高低点提醒
- 明确“不作为直接加减仓依据”

5. 次日观察点
- 2-3条可验证触发器（如：重新站回EXPMA30、结构消失、量能变化）
