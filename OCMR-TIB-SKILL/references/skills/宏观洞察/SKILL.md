# 宏观洞察技能

## 适用触发词
- `跑一下宏观洞察`
- `宏观怎么看`
- `给我宏观定价层结论`

## 核心职责
- 仅基于宏观变量输出定价层判断。
- 强制使用“证据数据 -> 推理链 -> 结论”格式。

## 输入要求
- 必需输入：`macro_liquidity_YYYYMMDD.json` 或同等宏观数据。
- 可选输入：当日市场新闻（仅补充背景，不替代宏观变量）。

## 执行步骤
1. 校验宏观数据字段与时间戳。
2. 生成宏观边际变化看板（5日/20日或本期/上期）。
3. 构建传导链：宏观变量 -> 折现率/风险偏好 -> 资产风格。
4. 输出结构化结论并标注证据来源。

## 输出格式
1. 宏观温度计
2. 边际变化看板
3. 流动性传导判断
4. 结论与证伪条件

## 参考资料
- 主参考：`../../p19-macro-insight.md`
- 执行标准：`../../../../RUNBOOK.md`

## 质量门禁
- 不允许用盘面涨跌替代宏观变量分析。
- 每项核心结论需给出对应数据证据。
- 缺关键变量时必须显式标注缺失。

## 失败处理
- 数据过旧或缺失时，输出“宏观数据不完整”并列出缺口。
- 变量冲突时，先列冲突变量，再给权重判断。
