# 盘面每日复盘技能

## 适用触发词
- `跑一下盘面复盘`
- `做今日复盘`
- `跑一下盘面解读`

## 核心职责
- 以`上证指数`作为唯一主判据输出复盘结论。
- 以`深证成指/创业板指/上证50/沪深300/中证500`做同步或背离校验。
- 按“趋势为王，结构修边”输出明确的总仓位动作建议。

## 指标口径（固定）
- 趋势主图（日线）：
  - 短期趋势：EXPMA(30)
  - 长期趋势：EXPMA(144)
- 趋势距离判定：
  - `|EXPMA30-EXPMA144|/EXPMA144 <= 2%` 视为“短长趋势很近”
- 结构副图（MACD）：
  - 周期：60m、90m、120m、日线
  - 需提示：钝化、结构、钝化消失、结构消失
  - 仅“结构形成”参与加减仓判断；钝化本身只提示不触发动作
  - 不要求四周期必须共振；有共振需单独标注“共振加强”
  - 结构有效影响窗口：对应级别后续24根K线
- TD9：
  - 仅作提示，不直接触发仓位动作
  - 周期：月线、周线、日线、120m、90m、60m

## 仓位动作规则（总仓位口径）
- 默认使用总仓位（100%基准）给出“减X层”。
- 无结构时：
  - 破短期趋势（EXPMA30） -> 减4层
  - 再破长期趋势（EXPMA144） -> 追加减2层（累计减6层）
- 结构形成时：
  - 价格靠近趋势且短长趋势很近（<=2%） -> 视为高威胁，减6层
  - 价格靠近趋势但未触发高威胁 -> 减4层
  - 价格远离趋势 -> 可不减（仅风险提示）

## 输出格式（每日固定）
1. 趋势状态（日线）
2. 结构状态（60m/90m/120m/日线）
3. 仓位动作建议（明确减几层，按总仓位）
4. TD9提示（月/周/日/120/90/60）
5. 次日观察点（2-3条）

## 质量门禁
- 必须先给上证指数主结论，再给其他指数同步校验。
- 每个仓位动作必须写明触发条件（趋势破位/结构形成/趋势距离）。
- 不得把TD9当作直接加减仓信号。
- 必须显式标注“结构是否共振”与“结构影响剩余周期”。
